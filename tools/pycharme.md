## 代码重构快捷键

序号	|快捷键|	作用
-- | --| --
1|	F5 |	复制文件
2|	F6 |	移动文件
3|	SHIFT + F6 | 重命名
4|	ALT + DELETE |	安全删除
5|	CTRL + F6 | 改变函数形式参数
6|	CTRL + ALT + M | 将代码提取为函数
7|	CTRL + ALT + V| 将代码提取为变量
8|	CTRL + ALT + C | 将代码提取为常数
9|	CTRL + ALT + F | 将代码提取为字段
10|	CTRL + ALT + P | 将代码提取为参数
